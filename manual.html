1. Primeros Pasos

Al abrir el sistema (index.html), verás una interfaz moderna y limpia diseñada para la eficiencia. La navegación principal se encuentra en la parte superior.

  • Navegación: Usa las pestañas superiores para moverte entre secciones.
  • Acceso Rápido: Botones rápidos para "Modo Acceso" e "Ingresos".

2. Panel Principal

Visualiza el estado de tu gimnasio al instante con estadísticas actualizadas:

Miembros Activos

Total de socios registrados actualmente.

Ingresos del Mes

Dinero total recaudado en el mes vigente.

3. Miembros

Para registrar un socio, ve a "Miembros" y completa los datos (Nombre, DNI, Vencimiento). El sistema genera un ID automático.

Puedes buscar a cualquier miembro para editar sus datos o eliminarlo si es necesario.

4. Pagos y Renovaciones

Registra ingresos de dinero vinculándolos al ID del socio.

Magia en Renovaciones

Al elegir el tipo pago "Mensualidad", el sistema calcula solo:

  • Si ya venció, suma 1 mes desde hoy.
  • Si sigue activo, extiende 1 mes a partir de su vencimiento actual.

5. Control de Acceso

En "Buscar", ingresa el ID del socio para ver su estado:

ACTIVA (VERDE)
VENCIDA (ROJO)

Si está vencida, el sistema te alertará antes de permitir el ingreso.

6. Actividades

Define tus actividades (Yoga, Crossfit, etc.) y asigna socios a cada una para llevar un control de aforo.

7. Copias de Seguridad

ACCIÓN OBLIGATORIA

Los datos se guardan en tu navegador. Usa "Exportar JSON" cada semana y guarda el archivo en un lugar seguro (Google Drive, Pendrive).
