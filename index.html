          Universo gym: Sistema de Gestión

          Guía completa para la administración profesional de tu gimnasio.

          Descargar PDF Ver en Pantalla Completa

          1. Primeros Pasos

          Al abrir el sistema (index.html), verás una interfaz moderna y limpia diseñada para la eficiencia.

          • Navegación: Usa las pestañas superiores (Miembros, Pagos, Buscar, Actividades) para moverte por las distintas secciones.
          • Acceso Rápido: En la barra superior tienes botones para "Modo Acceso" (para que los socios pongan su DNI) e "Ingresos" (para ver quién entra).

          2. Panel Principal (Dashboard)

          Nada más entrar, verás tarjetas con estadísticas clave que se actualizan en tiempo real:

          Miembros Activos
          Total de socios registrados en el sistema.
          Ingresos del Mes
          Suma total de dinero recaudado en el mes actual.

          3. Gestión de Miembros

          Administra tu base de clientes desde la pestaña "Miembros".

          Agregar un Miembro: Rellena Nombre, Apellido, Teléfono, Email, DNI y Fecha de Vencimiento. El sistema asignará un ID único automáticamente.

          Usa el botón "Ver Todos" para listar, editar (icono azul) o eliminar (icono rojo) socios.

          4. Gestión de Pagos y Renovaciones

          La sección más importante para tu flujo de caja.

          Renovación Automática

          Si registras un pago de tipo "Mensualidad":

          • Si ya venció: Sumará 1 mes desde el día de hoy.
          • Si está activo: Sumará 1 mes a su fecha actual de vencimiento.

          ¡No necesitas editar la fecha manualmente!

          5. Control de Acceso (Ingresos)

          Verifica si un socio puede entrar:

          1. Busca al socio por su ID en la pestaña "Buscar".
          2. Verás una etiqueta: VERDE (ACTIVA) o ROJA (VENCIDA).
          3. Pulsa "Registrar Ingreso" para guardar la entrada.
          Nota: Si alguien está vencido, el sistema te preguntará si deseas permitir un "Ingreso Excepcional".

          6. Gestión de Actividades

          Organiza tus actividades grupales:

          • Crea actividades con nombre, descripción y horario.
          • Inscribe socios asociando su ID a una actividad específica.
          • Controla el aforo y asistencia fácilmente.

          7. Copias de Seguridad

          ¡MUY IMPORTANTE!

          Tus datos se guardan localmente en el navegador. Para no perderlos:

          1. Usa el botón "Exportar JSON" regularmente.
          2. Guarda el archivo descargado en la nube o un pendrive.
          3. Si cambias de PC, usa "Importar JSON" para restaurar todo.
